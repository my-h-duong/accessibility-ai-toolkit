# Accessibility Findings

This folder contains reusable finding assets for consistent accessibility review output.

Findings should be:

- evidence-based
- reference-backed
- structured consistently

## Contents

- [Finding template](./templates/finding-template.md)
- [Severity and confidence guide](./severity-confidence.md)
- [Example: non-semantic interactive element](./examples/non-semantic-interactive-element.md)

## Folder Structure

- `templates/` contains canonical finding templates and authoring aids.
- `examples/` contains example findings written against the repository's evidence model.

Examples should stay generic, project-neutral, and reusable across review contexts.
