# Persistent Help Text

## Problem

Users need instructions to successfully complete a form field.

## Preferred Solution

Display help text directly below the field label.

## Avoid

Tooltip-only instructions.

Placeholder-only instructions.

## Reasoning

Instructions should be available when users need them.

Tooltip-only solutions require discovery.

Placeholders disappear when typing.

## Accessibility Impact

Improves usability for all users.

Reduces cognitive load.

Supports users with memory and attention challenges.

## Good Example

Email Address

We'll use this address for notifications.

[Input]

## Bad Example

Email Address (?) [Input]

Help text only appears when tooltip is activated.
